# iOS-translationApp
Using the model from Language Technology at the university of Helsinki to build a iOS translation app.


### Introduction
It is a iOS system App developed in Xcode. It is named "CC Trsanlation". It can translate 20 languages. The types of languages are in the following:


Arabic, Chinese, Danish, Dutch, English, Finnish, French, German, Greek, Hebrew, Hindi, Italian, Japanese, Norwegian, Polish, Portuguese, Spanish, Swedish, Turkish, Russian

*If you want to use or run, please input your api key on "NetworkManager.swift".*
